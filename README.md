## Introdução

O projeto Portfolio é uma aplicação web desenvolvida para exibir projetos em um portfólio pessoal. Ele possui três páginas principais: Home, Contact e Projects. A aplicação utiliza um modelo de dados para gerenciar informações dos projetos, componentes reutilizáveis que são adicionados ao HTML e uma interface de frontend.
